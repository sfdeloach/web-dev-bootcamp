137. Manipulating Style

    - tag.style.<access any css property here>
    - Not a good idea to use JS to manipulate style
    - It violates the seperation of concern principle
    - better to define a class in CSS and use:
        tag.classList.add("css-class") << adds a css class to element
        tag.classList.remove("css-class") << removes
        tag.classList.toggle("css-class") << the preferred method
